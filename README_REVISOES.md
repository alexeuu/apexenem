# ApexENEM • Somente revisões

## Como abrir

1. Extraia o ZIP inteiro para uma pasta.
2. Abra primeiro o arquivo `ABRIR_AQUI_LIMPA_CACHE.html`.
3. Se preferir, depois pode abrir `index.html` direto.

Esta versão usa um JavaScript empacotado (`js/app.bundle.js`), sem imports de módulos e com limpeza de cache, para evitar que os botões parem ao abrir localmente. Ela também limpa caches/service workers antigos.

# ApexENEM — Versão somente revisões

Esta versão foi ajustada para funcionar como um app de revisão espaçada.

## O que foi removido

- Botão e tela de organização semanal inteligente.
- Botão e tela de planejamento semanal.
- Arquivos JavaScript e CSS ligados à organização semanal.
- Textos do manual sobre organização semanal, fila semanal e modo emergência.

## O que continua funcionando

- Revisão espaçada por níveis R0, R1, R2... até Dominado.
- Fila diária de revisões.
- Filtros por matéria, prioridade, status e revisão.
- Calendário de revisões.
- Caderno de erros.
- Simulados.
- Backup, importação e exportação de progresso.
- PWA/app instalável.
