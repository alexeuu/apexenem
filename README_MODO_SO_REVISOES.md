# ApexENEM — Modo só revisões

Esta atualização adiciona um botão temporário para desativar o cronograma inteligente sem apagar nenhum dado.

## O que muda

- Novo botão no topo: **Usar só revisões**.
- Quando ativado, o site esconde os botões **Cronograma** e **Plano da semana**.
- O sistema de revisão espaçada continua funcionando normalmente.
- O cronograma salvo não é apagado. Se quiser voltar depois, clique no mesmo botão para reativar.

## Por que isso foi feito

O cronograma atual estava gerando muitos blocos e ficando irreal para o uso diário. Este modo permite continuar usando o ApexENEM apenas como app de revisões enquanto o cronograma é ajustado com calma.
